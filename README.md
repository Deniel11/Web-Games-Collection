# 🎮 Classic Web Games Collection

A small collection of classic browser games built with vanilla HTML, CSS, and JavaScript.

## 🕹️ Included Games

### 🪓 Hangman
A classic word-guessing game. Try to guess the hidden word one letter at a time. If you make too many mistakes... you lose!

- Built with: HTML, CSS, JS
- Features:
  - Random word generation
  - Keyboard-style letter input
  - Difficulty levels
  - Visual feedback for correct/incorrect guesses

### 💣 Minesweeper
Find all the mines without triggering any. Each number tells you how many bombs are touching that cell.

- Built with: HTML, CSS, JS
- Features:
  - Classic Windows-style design
  - Timer and mine counter
  - Difficulty settings
  - Right-click to place flags (long-press on mobile)

### 🔢 Sudoku
Fill the grid with numbers from 1 to 9 so that each number appears only once in each row, column, and 3×3 box.

- Built with: HTML, CSS, JS
- Features:
  - Interactive number grid
  - Auto-check for rule violations
  - Predefined puzzles

## 📱 Mobile Friendly

All games are responsive and optimized for both desktop and mobile screens.

## 🚀 Getting Started

To play the games locally:

1. Clone this repository:
```bash
git clone https://github.com/yourusername/web-games.git
cd web-games
```
2. Open the `index.html` file in your browser:
```bash
open index.html
```

Or simply host the folder using a static server (like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VSCode).
